Negrito / Itálico

Nesta frase, podemos usar o Itálico usando a tag <I>.

Nesta frase, podemos usar o Negrito usando a tag <B>.

Negrito / Ênfase

Nesta frase, podemos utilizar também o Negrito usando a tag <STRONG>.

Nesta frase, utilizamos também a Ênfase usando a tag <EM>.

Marcando texto

Podemos marcar um texto usando a tag <MARK>.

Letras Grandes e Pequenas

Nesta frase, podemos intriduzir textos grandes usando a tag <BIG>.

Nesta frase, podemos também adcionar textos pequenos usando a tag <SMALL>.

Texto Deletado

Nesta frase, podemos adcionar textos deletados ussando a tag <DEL>.

Nesta frase podemos também inserir textos usando a tag <INS>.

Textos Sobrescritos

Podemos usar tanto em formulas quanto em textos do tipo X3+2+4. No texto
Xelevado+N

Podemos também usar em textos ou formulas do tipo H2O. Ou do tipo
NcontidoN.
